#
- pwd 看你当前所在目录
- cd change directory
- ls 列出目录里的文件
- ls -la 列出全部文件(l=log,a=all)
- mkdir 创建目录(-p 级联创建)
- rm 删除目录或文件
- rm -rf www 递归强行删除目录及子目录 (r- recursive,f-force)
- cat 查看文件内容
- mv 移动
- cp 拷贝
# vi
- i 输入编辑模式
- esc 
- :wq 保存文件并退出
-  :wq!  强制保存并退出
- ：q 直接退出不保存修改
- / 搜索
- n 查找下一处
- N 上一处
- ctrl + f 向后翻页
-  ctrl +b 向前翻页
# 路径
- / 表示系统根目录
- ~ 当前所登录用户的主目录
- . 当前目录
- ../上一级目录
